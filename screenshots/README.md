## Cara Menambahkan Screenshot:

1. Buka aplikasi di browser
2. Ambil screenshot (Windows: Win+Shift+S, Mac: Cmd+Shift+4)
3. Simpan dengan nama sesuai list di atas
4. Paste ke folder ini
5. Screenshot akan otomatis muncul di README.md

## Tips Screenshot Bagus:

- Gunakan resolusi 1920x1080 atau lebih tinggi
- Crop hanya bagian aplikasi (tanpa browser bar)
- Gunakan data sample yang realistis
- Pastikan UI dalam kondisi bersih (no error messages)
- Gunakan mode light/dark yang konsisten
